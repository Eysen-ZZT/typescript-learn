

### 基础类型

##### 1. 数字类型\<Number>

```ts
let a: number;
a = 233;

let a = 233;

let a: number = 233;
a = '233'; // Argument of type 'string' is not assignable to parameter of type 'number'.

// 如果变量的声明和赋值是同时进行的，ts可以自动对变量进行类型检测
```

##### 2. 字符类型\<String>

```ts
let a: string;
a = 'hello'
```

##### 3. Boolean类型

```ts
let a:boolean = true;
```
